A general-purpose **agent harness** that gives LLM agents a **CLI-based computer** that enables them to complete tasks the way humans do, coworking with humans along the way.

**Vendor-agnostic by design**. OpenAgent isn't coupled to any specific LLM provider or proprietary features. The goal: bring powerful agentic capabilities to open-weight LLMs.

### Core Philosophy

1. **Give the agent a computer** (via the terminal)
2. Simplicity First
3. Testability, Composability, Minimal Dependencies (from [Adam Wolff](https://github.com/wolffiex/qcon-2025-ai-speed))
