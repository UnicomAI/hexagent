# 联网调研指南 (Web Search Mode)

**注意：** 这是一个从零创建 PPT 的可选分支。
- 如果用户**已经上传了相关素材**，请**优先处理并使用用户的素材**，而不是使用此联网调研模式。
- 如果用户没有提供足够素材或明确要求从零调研，才使用此模式。

## 工作流：调研 -> 压缩 -> 重新加载

当需要联网搜索素材时，请严格遵循以下步骤：

### 1. 联网搜索数据与材料
在开始构思 PPT 之前，先联网搜索相关主题的材料和**具体的数据**。

**⚠️ 工具使用优先级（必须严格遵守）：**
1. **优先使用内置联网工具**：如果环境中存在直接的 WebSearch 或类似的内置联网工具，请优先调用。
2. **其次使用 MCP 方式**：如果内置工具不可用，请尝试使用提供的网络搜索 MCP 工具。
3. **自己编写临时 Python 脚本**：由于本 Skill 不提供现成的联网搜索脚本，如果上述方式均不可用，请自行编写一个简单的 Python 脚本来完成网页内容获取。

*引导要求*：调研的内容必须包含**具体的数值、统计数据或事实支撑**，不能全是空洞的描述。

### 2. 生成调研报告 `research.md`
将搜索并整理好的内容写入一份 `research.md` 文档。
内容应结构化，涵盖 PPT 可能需要的背景、核心观点和关键数据。

### 3. 上下文压缩与重新加载
由于搜索过程可能消耗大量上下文，写完 `research.md` 后，你需要进行**上下文压缩**。
- 清理或总结冗长的搜索历史和中间过程。
- **重新加载**本 `pptx` skills，继续进行下一步（即生成大纲和制作幻灯片）。
